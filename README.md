# PRODIGY_WDS_01
Internship Task 1
World Population Data Visualization using TableauOverview

- This project focuses on data visualization and basic exploratory analysis of global population data using Tableau. The dataset used is the World Bank Total Population Dataset, which provides yearly population statistics for countries around the world.

The goal of this project is to visualize population distribution across countries and identify patterns using charts such as bar charts and histograms.

Dataset :

The dataset was obtained from the World Bank population database.

It contains population data for multiple countries across several years.

Important Columns ->

Country Name – Name of the country

Country Code – Country identifier

Indicator Name – Population indicator

Indicator Code – Indicator code

Year Columns (1960–2022) – Population values for each year

Project Objectives :-

Import and explore the population dataset.

Perform basic data cleaning.

Visualize population distribution using charts.

Identify patterns and trends in global population data.

Data Cleaning

Before visualization, the dataset required minor cleaning:

Removed the metadata rows at the top of the CSV file.

Ensured columns were correctly recognized by Tableau.

Selected a specific year (2022) for analysis.

Renamed fields for better readability.

Visualizations Created
1. Population by Country (Bar Chart)

A bar chart was created to show the population of different countries for a selected year.

Insights:

Countries like India, China, and the United States have the highest populations.

Smaller countries have significantly lower population values.

2. Population Distribution (Histogram)

A histogram was created to show the distribution of countries across population ranges.

Insights:

Most countries fall into lower population ranges.

Only a few countries have populations above 500 million.

Tools Used -

Tableau

Microsoft Excel (for basic dataset cleaning)



Conclusion

This project demonstrates how data visualization tools like Tableau can help explore and understand global population trends. By creating charts such as bar graphs and histograms, we can easily identify population patterns and compare countries.
